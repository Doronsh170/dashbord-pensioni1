# דשבורד חשיפות חיסכון v3.55

גרסה זו מתקנת את סינון היצרן לאחר בחירת חברה:
- בחירת כלל מסננת לפי שדות החברה המנהלת/החברה האם בלבד, ולא לפי המילה "כללי" בשם המסלול.
- בחירת מור מסננת לפי היצרן מור בלבד, ולא לפי המילה "מורים" בשם קופות אחרות.
- נשמר סדר תוצאות מועדף: כללי, עד גיל 50, מניות, ואז יתר המסלולים.
