# דשבורד חשיפות חיסכון v3.57

גרסה זו מבוססת על v3.56 ומעדכנת את מסך הפתיחה לעיצוב פרימיום נקי יותר:
- Hero ממורכז עם מיתוג, פרטיות, מקור נתונים ונתוני מאגר קצרים.
- בחירת סוג מוצר באמצעות שלושה כרטיסים ברורים.
- שמירה על זרימת החיפוש הקיימת: מוצר → סוג → חברה → קופה.
- ללא שינוי בחישובים, בקובץ הנתונים או בסימולציה.
