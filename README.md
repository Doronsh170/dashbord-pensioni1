# דשבורד חשיפות חיסכון

גרסה מעודכנת לפי הקובץ המעוצב החדש.

## מה כלול

- שורת חיפוש בלבד, ללא ריבועי סינון.
- ללא השורה העליונה עם הלוגו והטקסט באנגלית.
- שדה יתרה לכל קופה / קרן / פוליסה.
- שדה דמי ניהול מצבירה לכל מוצר.
- דמי הניהול בדשבורד מבוססים רק על מה שהמשתמש מזין ידנית.
- אם לא הוזנו דמי ניהול באחד המוצרים עם יתרה, הדשבורד מציג "לא הוזן" במקום ממוצע מטעה.

## הרצה מקומית

```bash
python -m http.server 8000
```

ואז לפתוח בדפדפן:

```text
http://localhost:8000
```
