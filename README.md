# דשבורד חשיפות חיסכון - v3.15

גרסה סטטית מוכנה להעלאה ל-GitHub Pages / Netlify.

## קבצים

- `index.html` - קובץ האתר הראשי
- `funds_unified.json` - נתוני גמל, פנסיה וביטוח מאוחדים
- `.nojekyll` - מונע עיבוד Jekyll ב-GitHub Pages
- `.gitignore` - קובץ התעלמות בסיסי

## עדכון מרכזי ב-v3.15

- נוספה הזנת דמי ניהול אישיים ליד שדה היתרה לכל מוצר.
- הדשבורד מחשב דמי ניהול אישיים משוקללים רק אם הוזנו דמי ניהול לכל המוצרים בעלי יתרה.
- אם חסר דמי ניהול במוצר מסוים, יוצג "לא הוזן" ולא ייעשה שימוש אוטומטי בדמי הניהול מתוך המאגר.
- דמי הניהול מהמאגר אינם משמשים לחישוב המרכזי, כדי לא להציג נתון מטעה.

## פריסה

אין צורך ב-build.

ב-Netlify:

```text
Build command: להשאיר ריק
Publish directory: .
```

ב-GitHub Pages:

```text
Settings → Pages → Deploy from branch → main → /root
```

## הערה מקצועית

הדשבורד מציג חשיפות ותשואות מסלול ציבוריות, יחד עם יתרות ודמי ניהול שהמשתמש מזין ידנית. הוא אינו מחליף ייעוץ פנסיוני או פיננסי.
