# דשבורד חשיפות חיסכון v3.50

גרסה זו מוסיפה חיפוש פנימי ממוקד לאחר בחירת סוג מוצר ויצרן:
- זרימה ראשית: סוג מוצר → חברה → חיפוש מסלול בתוך התוצאות → בחירת קופה.
- לאחר בחירת חברה מוצג שדה חיפוש פנימי לפי מסלול, גיל, מדד או מספר קופה.
- מוצגות תחילה עד 15 תוצאות עם כפתור "הצג עוד", כדי למנוע גלילה ארוכה.
- לא שונו חישובים, נתונים או סימולציה.
