# דשבורד חשיפות חיסכון v3.51

גרסה זו מוסיפה שלב משנה בזרימת הבחירה:
- גמל, השתלמות וגמל להשקעה: קופת גמל / קרן השתלמות / גמל להשקעה.
- ביטוח מנהלים או פוליסות חיסכון: ביטוח מנהלים / פוליסות חיסכון.

הזרימה המעודכנת:
סוג מוצר → תת-סוג מוצר → חברה → קופה / מסלול → יתרה ודמי ניהול → דשבורד.

לא בוצע שינוי בחישובי החשיפות, בדמי הניהול האישיים או בסימולציה על בסיס תשואות עבר.
